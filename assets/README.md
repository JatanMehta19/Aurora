# Assets

Drop image assets referenced by the project's `README.md` here.

## `ide-screenshot.png`

The main README embeds `assets/ide-screenshot.png`. To create it:

1. Launch the IDE: `python gui.py`
2. Load an example (e.g. **4 Functions & Recursion**) and press **F5 / Run**.
3. Capture the window and save it as `ide-screenshot.png` in this directory.

A 16:9 capture around 1280×760 (the default window size) looks best.
